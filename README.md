# Caesar Cipher (Python)

A Python console application that encrypts and decrypts messages using the Caesar Cipher technique. The user chooses whether to encode or decode a message and provides a shift value to transform each letter in the alphabet. Non-alphabet characters are preserved, making the program more user-friendly.

This project demonstrates core Python concepts such as functions, loops, conditional logic, list indexing, and importing code from multiple files.
